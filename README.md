# Gogin

Open-source full-stack Cloudflare Worker application skeleton.

Planned stack:

- TypeScript
- Cloudflare Workers
- Hono
- React
- Vite
- DaisyUI
- D1
- AI Gateway

## Structure

```txt
src/worker/      Worker runtime and API routes
src/web/         React frontend
migrations/      D1 migrations
public/          Static public assets
tests/           Worker and frontend tests
docs/            Project documentation
```
